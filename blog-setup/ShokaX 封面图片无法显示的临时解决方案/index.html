shokax封面图片无法显示的临时解决方案

首页博客构筑

# # 原因

目前新浪图床开启了防盗链，所以所有 cover 图片都无法正常加载，解决方法如下（二选一）:

  • 自托管封面图片
  • 使用 CSS 渐变模式

# 自托管

点我跳转到作者做的封面图片备份，将备份 clone 到本地或服务器上，剩下的和静态网站托管一样
修改 _images.yml 内的内容，参考此格式:

- http://example.com/img1.webp
- http://example.com/img2.webp
# ...

# 使用 CSS 渐变模式

此模式将文章封面修改为 CSS 渐变来解决此问题
修改主题 _config.yml

experiments:
  gradient: true # 使用CSS渐变作为文章封面
  fixedCover: "assets/cover.webp" # 当gradient为true时使用的主页面cover或者主界面的固定Cover(0.1.5+)
  # 上面的配置请设置为自己的背景图片

随后重新 hexo g 即可

参考链接：https://github.com/theme-shoka-x/hexo-theme-shokaX/issues/6
