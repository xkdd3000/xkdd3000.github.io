使用pnpm 安装 hexo-deployer-git

使用 pnpm 安装 hexo-deployer-git 非常简单。可以通过以下命令进行安装： pnpm add hexo-deployer-git --save# 解释 pnpm add ：这是 pnpm 用来安装依赖包的命令，相当于 npm install 。 hexo-deployer-git ：这是要安装的插件名称。 --save ：将依赖保存到 package.json 中，确保项目的依赖关系被记录。 安装完成后，你就可以在 Hexo 配置文件 _config.yml 中配置 hexo-deployer-git 来使用了。
